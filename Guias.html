DESCRIPCIÓN DE LAS GUÍAS

Descripción Guía 1

La Guía 1 tiene como objetivo que el estudiante sea capaz de desarrollar una comprensión sólida sobre los fundamentos del desarrollo web. Al finalizar, el estudiante podrá identificar la estructura básica de un documento web, entendiendo cómo se organizan y estructuran los elementos HTML. También será capaz de aplicar reglas de estilo CSS a los elementos HTML utilizando tanto estilos embebidos como hojas de estilo externas, mejorando la presentación visual de las páginas web. Además, aprenderá a utilizar selectores CSS básicos y las propiedades relacionadas con fuentes y textos, lo que permitirá la personalización efectiva de los contenidos textuales en un sitio web. A través de estos ejercicios, el estudiante adquirirá las competencias necesarias para comenzar a construir y diseñar páginas web funcionales y estilizadas.

Descripción Guía 2

La Guía 2 tiene como objetivo que el estudiante sea capaz de utilizar y combinar selectores CSS avanzados, lo que le permitirá un mayor control y precisión en el diseño de los elementos en la página. Además, comprenderá la utilidad de las pseudo-clases y pseudo-elementos CSS para agregar interactividad y estilos especiales en situaciones específicas. Finalmente, aprenderá a crear páginas web utilizando el Modelo de Cajas, identificando y trabajando con las partes clave de este modelo (margin, border, padding y content), lo que le permitirá estructurar y organizar el diseño de manera efectiva.

Descripción Guía 3

La guía 3 tiene como objetivo que el estudiante sea capaz de utilizar los diferentes tipos de posicionamiento CSS y diferenciarlos adecuadamente, lo que permitirá organizar y ubicar los elementos en la página con mayor precisión. También aprenderá a utilizar las propiedades CSS relacionadas con la visualización de los elementos, optimizando cómo se muestran en diferentes dispositivos. Finalmente, el estudiante será capaz de maquetar sitios web con diseños fijos y fluidos, dominando tanto estructuras estáticas como aquellas que se adaptan a diferentes tamaños de pantalla.

Descripción Desafío Práctico

Para este desafío práctico se le pide construir el sitio web de la Bolsa de Trabajo de ESEN. Para ello se le proporcionan cuatro páginas html denominadas index.html, ofertas.html, aplicar.html y consejos.html. Deberá aplicar todos sus conocimientos de CSS para conseguir que dichas páginas tengan el estilo final deseado (las imágenes con las capturas de cómo deberían quedar las páginas se proporcionan como recursos).

Descripción Guía 4

La Guía 4 tiene como objetivo que el estudiante aprenda a crear y estilizar formularios utilizando la diversidad de controles que ofrece HTML5. Además, comprenderá cómo utilizar correctamente la meta-etiqueta viewport para asegurar que las páginas web se visualicen adecuadamente en diferentes dispositivos. El estudiante también será capaz de definir correctamente las media queries para aplicar reglas de estilo en función del ancho del viewport del navegador, adaptando el diseño a diversas pantallas. Finalmente, aplicará los conocimientos adquiridos durante la asignatura para la construcción de páginas web responsivas, garantizando que estas se adapten a diferentes resoluciones de pantalla
