Datos Generales del Curso de Desarrollo Web

Descripcion del curso:Este curso es el primero en la secuencia de desarrollo web y cubre los fundamentos básicos de los lenguajes HTML, CSS y JavaScript, que son los elementos esenciales de cualquier aplicación web.
A través del dominio de los elementos del lenguaje HTML y el uso de los elementos de estilo en lenguaje CSS los estudiantes podrán diseñar sitios web atractivos y funcionales. Mediante el uso de los conceptos básicos y sintaxis de JavaScript, los estudiantes podrán crear aplicaciones interactivas de front-end e introducir contenido dinámico. Asimismo, serán capaces de comprender la arquitectura de las aplicaciones front-end, el rol de cada tecnología en la creación de aplicaciones modulares y conocerán las mejores prácticas en el diseño de estas. Finalmente, se discutirán algunas prácticas desventajosas en el desarrollo de software.

Elementos de competencia: a) Comprender y utilizar los diferentes elementos de lenguaje HTML para elaborar la estructura de una aplicación web. b) Comprender las propiedades y selectores utilizados en las hojas de estilo en cascada para aportar elementos de estilo a una aplicación web. c) Traducir las historias de usuario a elementos de diseño web que puedan ser implementados con la estructura de aplicación de una página. d) Desarrollar elementos que permitan manejar la interacción del sistema con el usuario de acuerdo con elementos de sintaxis moderna en Javascript o superconjuntos de dicho lenguaje.

Contenidos:HTML: Introducción a los elementos, semántica y atributos de HTML. CSS: Fundamentos de CSS, incluyendo selectores, propiedades y el modelo de caja. Maquetación web: Conceptos de posicionamiento, diseño básico, diseño responsivo con media queries y Flexbox. Frameworks responsivos: Uso de Bootstrap, su malla, utilidades y elementos. JavaScript: Fundamentos de JavaScript, estructuras de control, arreglos y objetos, funciones y eventos, y Programación Orientada a Objetos (POO). DOM y Javascript asíncrono: Manipulación del DOM, uso de JSON, almacenamiento en localstorage y sessionstorage, programación asíncrona (AJAX, APIs) y Firebase.
